# screenshots

리포트(`report.md`)에 들어갈 **본인 게임 캡처 이미지**를 이 폴더에 넣습니다.
(채점 규칙: 모든 리포트 항목은 본인 게임에서 캡처한 이미지로 설명해야 하며, 캡처 없는 항목은 0점)

## 캡처 방법
- 게임 실행 중 `H` 키로 HUD를 숨기면 깔끔한 화면을 캡처할 수 있음
- Windows: `Win + Shift + S` / Mac: `Cmd + Shift + 4`
- 저장한 png를 이 폴더에 넣고, `report.md`에서 `![설명](./screenshots/파일명.png)`로 참조

## 권장 파일명 (report.md의 placeholder와 일치)
| 파일명 | 내용 |
|---|---|
| `01_title.png` | 시작/타이틀 화면 |
| `02_overview.png` | 플레이 전경 |
| `10_gi_on.png` | GI 켜짐 (color bleeding 보임) |
| `11_gi_off.png` | GI 꺼짐 (`G`로 토글) |
| `12_dynamic_a.png` / `13_dynamic_b.png` | 다이버 이동 전/후 간접광 변화 |
| `14_camera.png` | 3인칭 카메라 |

> 파일명은 자유롭게 바꿔도 되지만, `report.md`의 참조와 반드시 일치시킬 것.
